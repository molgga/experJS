EXLoader Class

Defined in: experJS\net\EXLoader.js:18

(beta) 데이터를 load 할 수 있는 객체입니다.
데이터를 추가(addURL)하여 여러개의 데이터를 순차적으로 받는것을 지원합니다.
예제는 XmlLoader , ImageLoader 객체에서 참고하시기 바랍니다.

addURL

(
  • url
  • id
)
Void

Defined in experJS\net\EXLoader.js:158

load 할 데이터의 URL과 id를 지정합니다.
메소드 명에서 알 수 있듯 하나의 Loader 객체로 부터 여러개의 데이터를 추가하여 load 할 수 있습니다.
id 는 Loader 객체로 부터 로드가 완료된 후 데이터를 전달 받기 위해 필요합니다.

Parameters:

  • url String

    데이터 URL

  • id String

    데이터 id (데이터 id는 Loader 객체로 부터 데이터를 전달 받을 때 사용합니다.)
